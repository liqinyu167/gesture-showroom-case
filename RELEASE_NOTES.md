# v1.0.0

Gesture-interaction showroom case built around `MediaPipeCameraRouteTest.unity`.

## Assets

- `alispawn-gesture-showroom-full-project.zip`
  - Full Unity project archive.
  - Includes the embedded MediaPipe Unity package and gallery assets needed by the scene.
  - Recommended for Unity users who want to download, unzip, and run the case.
- `alispawn-gesture-showroom-source.zip`
  - Lightweight source archive.
  - Recommended for reviewing the scene scripts and interaction architecture.

## How to use

1. Download `alispawn-gesture-showroom-full-project.zip`.
2. Unzip it.
3. Open the extracted folder with Unity `2022.3.62f2c1` or a compatible Unity 2022.3 LTS editor.
4. Open `Assets/Scenes/MediaPipeCameraRouteTest.unity`.

The interaction principle diagram is included in `docs/interaction-principle.png`.
